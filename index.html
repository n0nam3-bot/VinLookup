How it works

1) VIN is decoded through the public NHTSA API to get year, make, model, and engine specs.
2) Decoded specs are shown in a clean table — only relevant fields like engine config, valve train, and turbo.
3) Compatible parts are generated by AI for your exact vehicle, grouped by category and quality tier (Economy / Standard / Premium).
4) If you load a local CSV/JSON catalog, those matched parts appear above the AI suggestions.
