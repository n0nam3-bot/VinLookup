NHTSA vPIC is a VIN-decoding source, not a complete OEM parts catalog. It can give you vehicle basics and many additional characteristics, but exact part-number inventories require a catalog source or your own parts database. That is why the page supports a local catalog file. ([nhtsa.gov](https://www.nhtsa.gov/cars/rules/manufacture?utm_source=chatgpt.com))

How it works

1) VIN is decoded through the public NHTSA API.
2) More than just year/make/model is shown when the API returns it.
3) If you load a local catalog, the page can filter exact parts rows by vehicle and keyword.
4) If no catalog is loaded, the page creates OEM/search shortcuts instead of pretending to know every part.
